# GEP

这是使用 C++ 实现的 GEP 算法，也就是 “ 基因表达式编程 ” 。编译并执行程序需要有可以支持 C++ 11 或以上版本的编译器，我本地编译采用的是 GCC 10 和 CMake 3.20 。

以我本地编译的方法为例：

    $ cd build
    $ cmake ../src
    $ cmake --build .

顺利的话，`build`目录里面会生成可执行程序`GEP.out`。如果采用其它的编译器或者编译套件，也可以简单地把`src`以及`src`下的子目录都设为搜索目录，然后执行编译即可。
